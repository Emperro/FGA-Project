FGA

Program Fresh Graduate Academy (FGA) merupakan program pelatihan bidang TIK yang terdapat 2 pelatihan di Pontianak yakni Big Data Analytics dan Cloud Computing.

Big Data Analytics

Big data analytics atau analisis big data adalah proses pengujian set data yang besar untuk menemukan pola yang tersembunyi, korelasi yang tidak diketahui, tren pasar, preferensi pelanggan dan informasi bisnis lainnya yang berguna.

Cloud Computing

Komputasi awan (cloud computing) adalah teknologi yang menjadikan internet sebagai pusat pengelolaan data dan aplikasi, di mana pengguna komputer diberikan hak akses (login).

Digipreneur

Kelas pendamping atau add-on yang akan didapat oleh peserta FGA yaitu kewirausahaan digital (Digipreneur) dan Soft Skills Development.
